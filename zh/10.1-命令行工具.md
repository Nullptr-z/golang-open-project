# 命令行工具

## urfave/cli 命令行

<https://github.com/urfave/cli>

```sh
go get github.com/urfave/cli
```

这个命令行可以生成目标 cli

```go
func main() {
  app := cli.NewApp()
  app.Name = "greet"
  app.Usage = "fight the loneliness!"
  app.Action = func(c *cli.Context) error {
    fmt.Println("Hello friend!")
    return nil
  }

  app.Run(os.Args)
}
```

也可以支持参数，标识等等，具体查看文档

## mkideal/cli 简单命令行
