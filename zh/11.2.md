# WebSokcet

## gorilla/websocket

```sh
go get -u -v github.com/gorilla/websocket
```

特性

- 支持接收分段消息
- 可发送关闭消息
- 支持心跳消息
- 支持获取消息数据类型
- 可限制收到的最大消息长度
- 支持io.Reader读取消息
- 支持io.WriteCloser写消息

仓库地址

https://github.com/gorilla/websocket

使用范例

https://github.com/gorilla/websocket/tree/master/examples
